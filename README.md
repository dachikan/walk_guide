# Walk Guide - 視覚障害者向け歩道案内「ロボット大脳」アプリ

視覚障害者のための音声誘導歩行支援アプリです。カメラで前方の状況を分析し、安全な歩行をサポートします。

## 主な機能

- **音声による状況案内**: 前方の状況を日本語で音声案内（10秒おき）
- **短期記憶機能（海馬）**: 10秒前と現在の画像を比較し、「電柱が近づいた」「信号が青に変わった」等の変化を通知
- **マルチAI対応**: Google Gemini, Claude 3.5 Sonnet, ChatGPT (GPT-4o mini) を切替可能
- **音声コマンド認識**: 「景色」「説明」「クロード」「ジェミニ」「GPT」などの音声操作に対応
- **通信エラー詳細通知**: エラー発生時に、「通信エラー」だけでなく具体的な原因を後出しで説明
- **生存確認ハートビート**: 4秒間隔のバイブレーションでアプリが正常動作していることを常時通知（OS設定：画面ロックON、タッチバイブON推奨）

## バージョン履歴
- **v0.0.8+1**: ステップ4〜6（通信エラー詳細通知、マルチAI、短期記憶）の実装、UTF-8文字化け解消

## 使用技術
- **Flutter**: 3.11.0
- **AI**: Gemini 2.5 Flash Lite, Claude 3.5 Sonnet, GPT-4o mini
- **音声**: speech_to_text, flutter_tts
